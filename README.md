# PedePizza | Chatbot

## Objective

To sell pizzas in an easy way with the App. Easy for the customer to order, easy for the company to sell.

## Host

Access the webApp: https://pedepizza.netlify.app/

- JavaScript
- CSS3
- HTML
- Figma
